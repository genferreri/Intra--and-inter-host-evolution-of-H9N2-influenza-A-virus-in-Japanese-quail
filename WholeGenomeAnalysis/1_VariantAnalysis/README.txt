The script GATK-LoFreq_VariantAnalysis_SLURM.sh performs variant analysis under GATK 
specifications. As variant caller uses LoFreq which has been designed for virus population 
analysis. The scripts utilises the raw fastq files along with the reference in fasta format.
The references needs to be named as "WF10_curated_varpatch.fa". This was designed to run in
the sapelo 2 computer cluster at the University of Georgia.
